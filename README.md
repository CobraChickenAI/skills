# Cobra Chicken AI Skills

A collection of [Agent Skills](https://agentskills.io) for AI assistants.

## Skills

| Skill | Description |
|-------|-------------|
| **[coherent-commit](coherent-commit/)** | Meaning-first git commits -- plain language, one file at a time, readable by humans and machines |
| **[porkbun](porkbun/)** | Domain management via the Porkbun API -- DNS, SSL, forwarding, and hosting blueprints |

## Installation

### Claude Code

```bash
/plugin marketplace add CobraChickenAI/skills
```

### Manual

Copy the relevant skill folders into your project's `.github/skills/` or `.claude/skills/` directory.
